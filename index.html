Bingo Engine — C++17 → WebAssembly
Motor completo de juego de Bingo desarrollado en C++17 y compilado a WebAssembly con Emscripten. Implementa la lógica completa: selección de apuesta, cartón, bombo de 30 bolas, bolas extra, freegames (25% de probabilidad) y fase de doble o nada. La API de C++ se expone a JavaScript mediante funciones EMSCRIPTEN_KEEPALIVE, con estado de juego serializado como JSON. Persistencia mediante localStorage (historial de partidas).
